Script Selenium untuk mensinkronisasi data satu PT dari situs PDDikti ke database, mencakup:

        ├── scrape_detail_prodi(detail_url)
        │   ├── ekstrak profil (telepon, email, website, tanggal berdiri, SK, deskripsi)
        │   ├── _click_tab("Tenaga Pendidik")
        │   │   └── _read_dosen_paginated()   → loop halaman dosen
        │   └── _click_tab("Mahasiswa")

### Iterasi 2 — Link detail prodi tidak ditemukan di HTML

**Masalah:** Kolom "Nama Program Studi" tidak mengandung tag `<a>` dalam HTML — link bersifat dinamis (muncul saat event hover/klik menggunakan JavaScript router Vue/React).

**Investigasi:** Cek innerHTML kolom → teks plain tanpa link. Cek semua `<a>` setelah hover → masih kosong. Coba klik cell → URL browser berpindah ke `/detail-prodi/...`.

**Solusi:** Klik `cells[1]` (kolom nama prodi), tunggu 6 detik, capture `driver.current_url`, lalu `driver.get(pt_url)` untuk kembali ke halaman PT. Diulang untuk tiap prodi aktif.

```python
cells[1].click()
wait(6, f"navigasi detail {kode}")
detail_url = driver.current_url
kode_to_detail[kode] = detail_url
driver.get(pt_url)
```

**Efek samping:** Setiap prodi memerlukan 1 round-trip ke PT page (buka PT → klik prodi → kembali PT). Untuk PT dengan banyak prodi ini menambah waktu, tapi diperlukan karena URL dinamis.

### Iterasi 3 — `MoveTargetOutOfBoundsException` pada ActionChains hover

**Error:**
```
selenium.common.exceptions.MoveTargetOutOfBoundsException:
  Move target (270, 773) is out of bounds of viewport dimensions (1366, 682)
```

**Sebab:** Elemen berada di luar viewport (di bawah layar). `ActionChains.move_to_element()` memerlukan elemen terlihat di viewport.

**Fix:** Scroll ke elemen sebelum interaksi:
```python
driver.execute_script("arguments[0].scrollIntoView({block:'center'});", cells[1])
time.sleep(0.5)
cells[1].click()
```
*(Catatan: akhirnya tidak menggunakan hover, langsung klik)*

### Iterasi 4 — `_read_table_rows` mengambil tabel yang salah

**Masalah:** Halaman detail prodi memiliki 3 tabel: (1) Riwayat Program Studi, (2) Dosen, (3) Mahasiswa kosong. Fungsi `_read_table_rows` selalu mengambil tabel pertama yang punya data, sehingga mengambil tabel Riwayat bukan tabel Dosen.

**Fix:** Tambahkan pengecekan header tabel — cari tabel yang mengandung kolom "NIDN":
```python
header_texts = [c.text.strip() for c in header_row]
if key_col not in header_texts:
    continue
```
`key_col` diambil dari `kolom[2]` yaitu `"NIDN"` untuk dosen.

### Iterasi 5 — `_read_mahasiswa_table` tidak menemukan data

**Masalah:** Tabel mahasiswa adalah tabel ke-3 di halaman (index 2), sedangkan fungsi iterasi dari tabel pertama dan berhenti saat menemukan data (tabel Riwayat juga punya data).

**Fix:** Cari tabel berdasarkan header row yang mengandung teks `"semester"` (case insensitive):
```python
headers = [c.text.strip().lower() for c in header_cells]
if "semester" not in headers:
    continue
```

### Iterasi 6 — Kontak (telepon/email/website) tidak memiliki label di HTML

**Masalah:** Di halaman detail prodi, nomor telepon, email, dan website ditampilkan sebagai teks plain tanpa elemen label atau key-value pair yang bisa diparse dengan `extract_after()`. Contoh HTML:
```
(0293)2950000
-
akesmuhtmg@gmail.com
akmt.ac.id
```

**Fix:** Gunakan regex pattern matching pada `body.text` line by line:
```python
# Telepon: hanya angka, tanda kurung, strip
if re.match(r'^[\+\(]?[\d\s\(\)\-]{7,20}$', line):
    telepon = line

# Email: mengandung @ dan format valid
if "@" in line and re.match(r'^[^@\s]+@[^@\s]+\.[^@\s]+$', line):
    email = line

# Website: domain tanpa spasi, bukan nama jalan
if "." in line and "@" not in line and re.match(r'^[a-zA-Z0-9][\w\-\.]+\.[a-zA-Z]{2,}(/.*)?$', line):
    if not re.search(r'(Jalan|jendral|Senayan|Jakarta|Kab\.|Prov\.)', line, re.I):
        website = line
```

## Catatan Teknis

- **URL PDDikti bersifat dinamis:** Setiap request menghasilkan URL berbeda (token di path). Tidak perlu disimpan ke DB.
- **Pagination dosen:** Script mengikuti halaman berikutnya hingga tidak ada tombol next (safety limit 30 halaman).
- **Data mahasiswa:** Hanya 3 semester terakhir yang diambil sesuai petunjuk dokumentasi.
- **Info_umum / ilmu_dipelajari / kompetensi:** Pada AKM Temanggung bernilai "-" di PDDikti (belum diisi). Script `clean_block()` mengabaikan nilai "-" sehingga field tetap kosong.
- **Selenium:** Firefox headless, binary `/snap/firefox/current/usr/lib/firefox/firefox`.
- **Database:** MySQL remote `biroti-ums.id`, credentials dari `.env` di root backend.
